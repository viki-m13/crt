What this does

For each stock, the model measures how far the price has fallen, where it sits inside its 1‑year range, and whether the move looks stock‑specific (not just “the market”). It then finds the K most similar historical days for that same stock and checks what happened next.

How it works
  1. Measure today. Drawdown, range position, volatility/volume stress, and market‑adjusted weakness.
  2. Find similar past days. Uses a feature‑distance match, balanced across market regimes.
  3. Summarize outcomes. 1Y / 3Y / 5Y forward returns from those similar days.
  4. Score + rank. Higher score means rebounds were historically stronger after similar setups.
How to read the numbers
  • Chance of gain = % of similar past days where the forward return was positive.
  • Typical = the median forward return (the “middle” outcome).
  • Downside (1 in 10) = the 10th percentile (a bad-but-plausible case).
  • N = count of similar past days used in the summary.
